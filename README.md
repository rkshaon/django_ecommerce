# django_ecommerce

## To run the project
1. Install python
2. Open Command Prompt
3. Write `git clone https://github.com/rkshaon/django_ecommerce.git` and press enter
3. Write `pip install -r requirements.txt` and press enter
4. Write `python manage.py runsever` and enter


## Features:
1. store
2. cart
3. checkout
